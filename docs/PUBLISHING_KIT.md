## GitHub Release description — v1.0.0

**J.A.R.V.I.S. v1.0.0 — Initial Release**

The first public release of J.A.R.V.I.S., a Python voice assistant for
Windows powered by Anthropic's Claude API.

**Highlights**
- Wake-word activation and natural voice command routing
- Claude-powered conversational fallback for open-ended questions
- Application launcher, website shortcuts, Google/YouTube/Wikipedia search
- System info, volume, screenshot, and lock-screen controls
- Power controls (shutdown, restart, sleep, sign out)
- File/folder creation and navigation, typing, clipboard, and window management
- API key now loaded securely from an environment variable instead of being
  hardcoded

**Full changelog:** see `CHANGELOG.md`

## LinkedIn project description

```
Built J.A.R.V.I.S., a voice-controlled AI desktop assistant for Windows using
Python and Anthropic's Claude API. The assistant listens for a wake word,
routes commands to native Windows actions (launching apps, controlling system
volume, taking screenshots, managing power states), and falls back to Claude
for open-ended conversation, coding help, and general Q&A. Implemented speech
recognition with the SpeechRecognition library, text-to-speech with pyttsx3,
and system automation with PyAutoGUI and psutil — all wrapped in a clean,
single-file command router with 25+ recognized command categories.
```
