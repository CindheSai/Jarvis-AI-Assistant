# Voice command reference

All commands are triggered after saying the wake word **"jarvis"** and waiting
for "Yes, Sir?".

## Applications
| Say | Result |
| --- | --- |
| "open notepad" | Launches Notepad |
| "open calculator" | Launches Calculator |
| "open paint" | Launches Paint |
| "open word" | Launches Microsoft Word |
| "open excel" | Launches Microsoft Excel |
| "open powerpoint" | Launches Microsoft PowerPoint |
| "open chrome" | Launches Chrome |
| "open firefox" | Launches Firefox |
| "open vs code" | Launches VS Code |
| "open file explorer" | Launches File Explorer |
| "open task manager" | Launches Task Manager |
| "open settings" | Opens Windows Settings |
| "open camera" | Opens the Camera app |
| "open spotify" | Launches Spotify |
| "open \<anything else>" | Attempts to launch it as a raw process name |

## Wikipedia
| Say | Result |
| --- | --- |
| "wikipedia \<topic>" | Reads a 2-sentence summary |

## System controls
| Say | Result |
| --- | --- |
| "system info" / "cpu" / "ram" / "memory" / "disk space" / "battery" | Reports CPU/RAM/disk usage |
| "screenshot" / "screen capture" | Saves a screenshot to the desktop |
| "lock screen" | Locks the PC |

## Power controls
| Say | Result |
| --- | --- |
| "shutdown" | Shuts down in 10 seconds |
| "restart" / "reboot" | Restarts in 10 seconds |
| "sleep" / "hibernate" | Puts the PC to sleep |
| "cancel" | Cancels a pending shutdown/restart |
| "log off" / "sign out" | Signs out of Windows |

## Volume
| Say | Result |
| --- | --- |
| "volume up" / "increase volume" / "louder" | Raises volume |
| "volume down" / "decrease volume" / "quieter" / "lower volume" | Lowers volume |
| "mute" | Mutes audio |
| "unmute" / "volume on" | Unmutes audio |
| "max volume" / "full volume" | Sets volume to maximum |

## Window controls
| Say | Result |
| --- | --- |
| "minimize" | Minimizes the active window |
| "maximize" | Maximizes the active window |
| "close" / "quit" / "exit" | Closes the active window (Alt+F4) |
| "switch window" / "alt tab" / "next window" | Switches windows |
| "goodbye" / "bye jarvis" / "exit jarvis" / "stop jarvis" / "shut down jarvis" | Exits the JARVIS program |
